# Test Mode Guide

## Overview

Test Mode allows you to quiz yourself on your flashcards with two different testing formats. It's perfect for checking your progress, preparing for exams, and identifying areas that need more study.

## Getting Started

### Minimum Requirements

- **Written Test**: At least 1 card
- **Multiple Choice**: At least 4 cards (for generating wrong answer options)

### Starting a Test

1. Navigate to the **Test** tab
2. Choose your preferred format:
   - **Written Test** - Type your answers
   - **Multiple Choice** - Select from options
3. Click to start

The test will automatically select up to 10 random cards from your collection.

---

## Test Formats

### Written Test

**How it works:**
- You see the front of a card (question)
- Type your answer in the text box
- Press Enter or click "Next Question"
- Continue until all questions are answered

**Grading:**
- Answers are case-insensitive
- Whitespace is trimmed
- Must match the back of the card exactly

**Best for:**
- Testing exact recall
- Practicing spelling
- Deep learning verification
- Academic preparation

**Tips:**
- Type carefully - spelling matters!
- Use the translation hint if provided
- Press Enter for quick navigation
- Take your time on each question

---

### Multiple Choice

**How it works:**
- You see the front of a card (question)
- Choose from 4 options (A, B, C, D)
- One correct answer, three random wrong answers
- Click your choice to move forward

**Option Generation:**
- Correct answer: The actual back of the card
- Wrong answers: Randomly selected from other cards
- All options shuffled randomly

**Best for:**
- Recognition practice
- Easier confidence building
- Quick assessment
- Beginners or casual review

**Tips:**
- Read all options before choosing
- Use process of elimination
- Trust your first instinct
- Don't overthink it

## Scoring System

### How Grading Works

**Written Test:**
- Case-insensitive comparison
- Extra spaces ignored
- Must match card's back text exactly
- No partial credit

**Multiple Choice:**
- Correct option selected = point
- Wrong option = no point
- Simple binary scoring

### Score Calculation

```
Score = (Correct Answers / Total Questions) √ó 100
```

**Example:**
- 8 correct out of 10 questions
- Score = (8/10) √ó 100 = 80%

### Performance Levels

- **90-100%**: Excellent! You've mastered this material
- **70-89%**: Good job! Minor review needed
- **50-69%**: Decent, but more study recommended
- **Below 50%**: Need more practice with these cards

## Study Strategies

### Using Tests Effectively

**1. Regular Testing**
- Test yourself weekly
- Track score trends over time
- Identify consistent problem areas

**2. Progressive Difficulty**
- Start with Multiple Choice
- Move to Written Test when confident
- Challenge yourself gradually

**3. Focus on Mistakes**
- Review incorrect answers carefully
- Study those specific cards more
- Retest until you get them right

**4. Pre-Exam Preparation**
- Take multiple tests before exams
- Simulate test conditions
- Build confidence through practice

**5. Spaced Testing**
- Don't test immediately after studying
- Wait a day or two
- Better measure of retention

## Common Questions

### Q: How many questions are on a test?
**A:** Up to 10 questions, or fewer if you have less than 10 cards.

### Q: Are the same cards selected each time?
**A:** No, cards are randomly selected each time you start a test.

### Q: Can I pause a test?
**A:** No, tests must be completed in one session. They only take 5-10 minutes.

### Q: What if I make a typo in written test?
**A:** Unfortunately, it will be marked incorrect. Spelling counts!

### Q: Can I see the test questions again?
**A:** Yes, after completion, you can review all questions and answers.

### Q: How do I improve my score?
**A:** Study weak cards in Study mode, then retest. Repetition is key!

### Q: Is my test data saved?
**A:** No, test results are not stored. Tests are for immediate assessment only.

## Tips for High Scores

### Written Test Tips

1. **Read the question twice** before answering
2. **Check your spelling** before submitting
3. **Use exact wording** from the card back
4. **Don't add extra words** or punctuation
5. **Stay calm** - you know this!

### Multiple Choice Tips

1. **Eliminate obvious wrong answers** first
2. **Look for context clues** in the question
3. **Trust your first instinct** usually
4. **Don't overthink** simple questions
5. **Use the translation** as a hint

## Troubleshooting

### "Multiple Choice requires at least 4 cards"
**Solution:** Create more cards or use Written Test format.

### "No cards available for testing"
**Solution:** Go to Create mode and add some flashcards first.

### Test feels too easy
**Solution:** Use Written Test instead of Multiple Choice, or create more challenging cards.

### Test feels too hard
**Solution:** Spend more time in Study mode first, or start with Multiple Choice format.

### Same cards appear repeatedly
**Solution:** This is random, but create more cards for variety.
